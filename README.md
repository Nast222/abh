# Weather Diary

**Автор:** [Фирсова]
**Язык:** Python 3

## Описание программы

Графическое приложение для ведения дневника погоды. Позволяет добавлять записи о погоде за определённую дату с указанием температуры, текстового описания и наличия осадков. Все данные сохраняются в файл `weather_data.json`. Доступна фильтрация записей по дате и температуре.

## Требования

- Python 3.x.
- Библиотека `tkcalendar` (для календаря).
  ```bash
  pip install tkcalendar
  ```

## Запуск приложения

1. Убедитесь, что установлен Python 3.
2. Скачайте или клонируйте репозиторий.
3. Установите зависимости:
   ```bash
   pip install tkcalendar
   ```
4. Запустите скрипт:
   ```bash
   python weather_diary.py
   ```

## Использование программы

### Добавление записи

1. Введите дату с помощью календаря.
2. Укажите температуру (число).
3. Введите описание погоды (например: *«солнечно»*, *«дождь»*).
4. Отметьте галочку «Осадки», если был дождь/снег.
5. Нажмите кнопку **«Добавить запись»**.
6. Данные автоматически сохранятся в файл `weather_data.json`.

### Фильтрация записей

- Введите дату в поле фильтра — отобразятся только записи за этот день.
- Введите температуру (например: `10`) — отобразятся записи с температурой выше 10°C.
- Можно использовать оба фильтра одновременно.
- Нажмите **«Применить фильтр»**.
- Для возврата к полному списку перезапустите приложение или удалите фильтры и снова примените.
